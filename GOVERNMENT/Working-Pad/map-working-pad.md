# Map - working-pad

LSWT 프로젝트의 진행 중 작업, 제안, 이슈, 검토 대기 문서를 보관하는 운영 작업 공간.

- 아직 정본화되지 않은 운영 지식은 이 계층에 둔다.
- 사용자 승인 또는 검토 후 durable한 절차는 `GOVERNMENT/Agents-Bylaws/`에 둔다.
- 보호해야 할 확정 원칙은 `GOVERNMENT/User-Constitution/`에 둔다.

## Contents

| Item | Role |
|---|---|
| `handoff/` | 대화/작업 세션 인수인계 문서. 색인: `handoff/map-handoff.md` |
| `idea-proposals/` | 구조·철학·방향 제안 백로그. 색인: `idea-proposals/map-idea-proposals.md` |
| `issues/` | 미해결/종결 이슈 기록. 색인: `issues/map-issues.md` |
| `staging/` | 정본 승격 전 검토 대기 문서. 색인: `staging/map-staging.md` |
| `map-working-pad.md` | This navigation file |

## Agent Instructions

- 새 작업 문서는 먼저 이 계층에 둔다.
- 결론난 절차 문서는 `GOVERNMENT/Agents-Bylaws/` map과 함께 갱신한다.
- 파일 추가, 이동, 종결 처리 시 해당 하위 map 파일을 갱신한다.
