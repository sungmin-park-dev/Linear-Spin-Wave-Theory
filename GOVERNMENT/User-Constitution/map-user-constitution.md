# Map - user-constitution

LSWT 프로젝트의 사용자 승인 정본 레이어.

- 보호해야 할 원칙, 정의, 확정 결정만 둔다.
- 에이전트는 사용자 승인 없이 이 레이어의 본문을 수정하지 않는다.
- 현재 LSWT에는 승인된 정본 문서가 없다.
