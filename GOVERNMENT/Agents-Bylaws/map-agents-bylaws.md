# Map - agents-bylaws

LSWT 프로젝트에서 에이전트가 따르는 절차, 검증 계획, 운영 가이드 레이어.

- 에이전트가 유지할 수 있는 절차성 지식을 둔다.
- 오래되었거나 검토가 필요한 문서는 `status: in-review`와 본문 설명으로 표시한다.
- 사용자 승인 정본은 `User-Constitution/`에 둔다.

## Contents

| Item | Role |
|---|---|
| `policies/` | LSWT 운영 문서 정책. 색인: `policies/map-policies.md` |
| `procedures/` | 검증·작업 절차 문서 |
| `templates/` | AAD-style 운영 문서 템플릿 사본. 색인: `templates/map-templates.md` |
| `map-agents-bylaws.md` | This navigation file |

## Agent Instructions

- 새 절차 문서를 추가하면 관련 map 파일을 갱신한다.
- 새 운영 문서를 만들 때는 먼저 `policies/`와 `templates/`의 대응 문서를 확인한다.
- 검토 필요 문서는 그대로 실행 기준으로 삼지 말고 먼저 현재 구조와 대조한다.
- 코드/물리 의도가 불명확한 절차는 사용자에게 확인한다.
